Reflection

• Explain the concept of Responsive Design. What does the term mean?
Responsive design is an approach in HTML and CSS that allows the developers to design their websites and applications for various devices such as cellphones, desktops, and tablets. By using responsive design, we can develop websites that automatically adjust the website’s content and sections to match the client-side screen size. All in all, the term responsive means to make that design adaptable and responsive across all the platforms. One way to do this is to use media queries which is a feature of CSS3 that allows us to add certain features and display the content of the website according to screen size, platform, and device resolution.

• How does responsive design relate to web technologies?
a responsive web design ensures that your site is equally practical on all types of devices and does not lose functionality with newer electronic devices coming to the market in the future. It ensures that you're ready for future technology, avoiding the need for a redesign every time a new device is introduced to the market. Because with a few adjustments, you can make your previous design adjustable to the new device's screen size. In these modern times, we cannot use static design anymore, we need to use this new approach if we want to keep up with the new technologies out in the market.
